# Техническое задание на проект: CinemaMood

## 1. Цель проекта
Создание веб-сервиса для подбора фильмов по настроению и доступному времени. 
Сервис помогает пользователям быстро выбрать фильм для просмотра, анализируя 
их текущее состояние и предлагая персонализированные рекомендации.

## 2. Роли пользователей
* **Гость:** Может использовать подбор фильмов по времени и настроению.
* **Авторизованный пользователь:** Может сохранять подборки и историю просмотров.
* **Администратор:** Полный доступ к админ-панели Django.

## 3. Модели данных
1. **Movie (Фильм):** title, description, duration, release_year, rating, poster_url
2. **Mood (Настроение):** name, slug - теги для фильмов (комедия, драма, вдохновляющее)
3. **UserSelection (История подбора):** selected_mood, available_time, search_timestamp, matched_movies (ManyToMany)

## 4. Ключевой функционал
* Подбор фильма по времени и настроению через форму
* Добавление фильмов через TMDB API
* Просмотр каталога с аналитикой (графики через Matplotlib)
* Администрирование через Django Admin

## 5. Внешние интеграции / Аналитика
* **Интеграция:** The Movie Database (TMDB) API для получения данных о фильмах
* **Аналитика:** Pandas для анализа данных, Matplotlib для построения графиков
